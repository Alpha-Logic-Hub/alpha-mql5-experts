# Alpha Logic Hub — AI Trading System Plan

Este paquete resume la idea conversada: construir una fábrica de estrategias de trading asistida por IA, no solamente un repositorio de código MQL5.

## Objetivo

Crear un sistema donde `AGENTS.md`, skills, subagentes, specs y protocolos trabajen juntos para diseñar, implementar, auditar y validar estrategias de trading con disciplina profesional.

## Cómo leer esto

1. `01-plan-maestro.md` — visión general y principios.
2. `02-skills-propuestas.md` — skills iniciales recomendadas.
3. `03-subagentes-propuestos.md` — roles de agentes y responsabilidades.
4. `04-flujo-operativo.md` — pipeline de idea a estrategia validada.
5. `05-agents-md-template.md` — base para un `AGENTS.md` enfocado en trading.
6. `06-roadmap.md` — orden recomendado para construirlo.
7. `07-soulzbtc-insights.md` — ideas extraídas del documento “How to Master Claude in One Weekend”.
8. `08-repo-audit-remediation.md` — plan para convertir el repo actual en una base confiable.

## Regla madre

> La IA puede proponer, investigar y construir; pero ninguna estrategia se acepta sin hipótesis medible, riesgo controlado, compilación, backtest reproducible y revisión crítica.
